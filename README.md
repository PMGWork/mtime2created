# mtime2created

ファイルの更新日時 (mtime) を、作成日時 (birthtime) と同じ日時に変更する Obsidian プラグインです。

> [!WARNING]
> このプラグインは **macOS 専用** です。macOS 固有の `SetFile` コマンドに依存しています。

## 機能

- 現在アクティブなファイルの更新日時を、そのファイルの作成日時と同じ日時に変更します。
- ファイルエクスプローラーやエディタ上の右クリックメニューからも実行できます。
- ファイルの作成日時を、更新日時として適用したい場合に便利です。

## 要件

- macOS
- Obsidian

## インストール方法

1. 最新のリリースから `main.js`, `manifest.json`, `styles.css` (もしあれば) をダウンロードします。
2. 保管庫 (Vault) の `.obsidian/plugins/` ディレクトリ内に `mtime2created` という名前のフォルダを作成します。
3. ダウンロードしたファイルをそのフォルダに配置します。
4. Obsidian を再読み込みし、設定でプラグインを有効にします。

## 使い方

1. 変更したいファイルを開きます。
2. コマンドパレットを開きます (`Cmd + P`)。
3. **"Sync mtime to created"** を検索して実行します。

または：

1. ファイルエクスプローラーなどで対象のファイルを右クリックします。
2. メニューから **"Sync mtime to created"** を選択します。

## 開発

リポジトリをクローンし、依存関係をインストールします:

```bash
npm install
```

開発ビルドを開始します (ウォッチモード):

```bash
npm run dev
```

本番用にビルドします:

```bash
npm run build
```
